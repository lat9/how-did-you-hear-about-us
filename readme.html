Prerequisites

This version of the plugin requires:

  1. A working copy of Zen Cart 2.1.0 or later.
  2. The site must use a PHP version greater than or equal to PHP 8.0.

Overview

This plugin creates a pull-down menu that prompts the customer to identify where they found out about the site. The field can be made either required or optional via the admin's Configuration :: Customer Details. The default choices for the pull-down menu are set via the admin's Localization :: Referral Source Choices tool. Also configurable is whether or not to include an Other selection in that dropdown.

The customer's choice is saved in the customers_info table for follow-on reports; see the admin's Reports :: Referral Sources.

Installation

Follow these steps to install the plugin:

  1. Unzip this distribution's zip-file; you'll see a zc_plugins directory, and inside it a HowDidYouHear directory, and inside that another directory with a version number.
  2. Copy the plugin's HowDidYouHear directory to your site's file-system's zc_plugins directory.
  3. Log into your Zen Cart admin and go to Modules :: Plugin Manager. Click on How Did You Hear About Us, and click Install, and confirm by clicking Install.
  4. You'll see a couple of new settings in the admin's Configuration :: Customer Details tab; proceed to the configuration step, see below.
Uninstalling

To uninstall the plugin:

  1. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  2. Locate the entry for How Did You Hear About Us, click the associated Un-Install button and confirm the removal.

Note: Uninstalling the plugin removes its configuration settings and its registration in the admin menus, but does not remove:

  1. The sources and sources_other database tables. To fully remove these tables, enter the following in the admin's Tools :: Install SQL Patches:
    • DROP TABLE sources; DROP TABLE sources_other;
  2. The customers_info_source_id field from the customers_info table. To remove this field, enter the following in the admin's Tools :: Install SQL Patches:
    • ALTER TABLE customers_info DROP COLUMN customers_info_sources_id

Configuring

The following additional settings are added to the admin's Configuration :: Customer Details:

  1. Display "Other" Referral option?: Set this to true to enable the insurance module; default true.
  2. Require Referral Source?: This setting controls whether the referral-source field is required (true, default) or not (false).

Use Localization :: Referral Source Choices to modify the default choices displayed in the plugin's pull-down menu on the storefront create_account form.
